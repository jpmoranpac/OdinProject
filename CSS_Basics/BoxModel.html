Box Model Elements

padding: space between box border and content

border: space between margin and padding

margin: space between border of this box and adjacent boxes. If two margins meet, the smaller is disregarded (margin collapsing).

Block vs Inline Elements

block: elements separate vertically

inline: element appear on the same line (e.g. anchor element). Generally, do not use padding or margin on these elements, as width and height are not respected (interpreted from content length) and padding overlaps other elements. inline-block respects both these properties.

inline-block: probably superannuated by flexbox

div and span: elements that are used for grouping or hooks. Div is block-level (divides into vertical slices), span is in-line.

Flexbox

Flex boxcheat sheet

Terminiology

Flex container: any element with "display: flex" on it.

Flex item: any item directly in a flex container.

Elements can be both a flex container and flex item.

Properties

The "flex" property (e.g. div { flex: 1 1 0; } or div { flex: 1; }) is shorthand for:

  • flex-grow: 1
  • flex-shrink: 1
  • flex-basis: 0

flex-grow: the relative size of each item within a group

flex-shrink: how much each item will shrink if the parent container isn't large enough. A flex-shrink of "0" will not shrink at all. Usually relevant when setting a specific width for an item, where the item will shrink to no smaller than the specified width.

flex-basis: the initial size for the item. Default is 0%, so it ignores any defined "width" property.

Practically, the most common flex properties are "flex-grow: 1" to evenly grow divs or "flex-shrink: 0" to prevent shrinking.

flex-direction: default is "row", can be set to "column".

When sorting by column, the height must be set. The default height for an empty div is 0, so the flex items would collapse without an explicit height set. This is opposed to width, which block-level elements default to the full width of the parent.

Alignment

justify-content: arrange along main axis. Options: flex-start, center, flex-end, space-between, space-around, space-evenly.

align-items: arrange along cross axis. Options: flex-start, center, flex-end, stretch, baseline

align-self: same as "align-items", but only applies to a single item rather than all children of a container.

gap: similar to margin, adds space between items

flex-wrap: off by default (enable via flex-wrap: wrap;), allows items to move onto a new row if there is insufficent space on the current row.

Tips

No matter how high flex-shrink is set, the min-width defaults based on the items' content. To override this, set min-width: 0;

Flexbox Examples

Combining multiple simple flex box elements: 
form {
        display: flex;
        align-items: flex-end;
        flex-wrap: wrap;
        gap: 16px;
      }
      .name {
        flex-grow: 1;
        flex-basis: 160px;
      }
      .email {
        flex-grow: 3;
        flex-basis: 200px;
      }
      button {
        flex-grow: 1;
        flex-basis: 80px;
      }
