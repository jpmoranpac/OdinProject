Box Model & Flexbox

Intro

To apply a style sheet, include the following in the HTML file:

<head><link rel="stylesheet" href="styles.css"></head>"

Alternatively, can include the CSS internal to the HTML. The advantage of separate styles is that is can be shared amongst multiple HTML files. These are included under a <style> tag in the <head> tag.

In-line styles are not recommended, but can be used to override existing rules. To use, add a <style> tag to the element.

Selectors

Universal (*): All elements

Type: All of a given type

Class: (.[classname classname ...]): All elements with the matching "class" attribute

ID (#[ID]): All elements with the ID. Use sparingly.

Grouping Selectors: comma-separated list

Chaining Selectors (.[classname].[classname]) (.[classname]#[ID]): Applies to elements that have both classname "class" attributes

Descendant Combinator (.ancestor .child): Elements that have the "child" class and have a parent with the "ancestor" class

Properties

Common Properties

color: text colour

background-color: background colour

Colour values: #HEX, rgb(0, 0, 0), hsl(0, 0%, 0%)

CSS Legal Colour Values

font-family: comma-separated list of fonts. Specific ("Times New Roman") or generic ("serif")

font-size: e.g. 22px

font-weight: e.g. 700 (bold font), between 1-1000

image: height and weight: px values or "auto"

Cascade

When one element has conflicting properties, the following heirarchy is applied:

    • ID selector
      Class selector
      Type selector

    • Within a level of priority, the more specific rule takes precedence. The final tiebreaker is the order: the last defined rule wins.

    Additionally, typography properties are inherited by decendant elements.
