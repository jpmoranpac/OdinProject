    • Document Object Model

    JavaScript to modify HTML documents mostly need to be run after the relevant HTML has been created. This means scripts must be delcared at the end of the HTML document or loaded with the defer flag.

    In JavaScript, can select objects through CSS-style selectors or relational selectors.

    const container = document.querySelector("#container");
    const display = container.firstElementChild;

    Query selector:

    • element.querySelector(selector): returns a reference to the first match.
      element.querySelectorAll(selectors): returns a NodeList containing references to all matching selectors. Note that a NodeList isn't an array (can be converted via Array.from()).

    • With a reference to an element, can access the style via .style.property or .setAttribute.

    Toggling classes: div.classList.toggle("class")

    Text content: div.textContent = "Text"

    Element creation: const div = document.createElement("div");

    Append elements: parentNode.appendChild(Node) or parentNode.insertBefore(Node, referenceNode)

    Remove elements: parentNode.removeChild(child)

    Events

    The preferred way is through adding an Event Listener to an object: btn.addEventListener("click", alertFunction). The event listener class passes an e event argument.

    Adding events to a NodeList:

            const buttons = document.querySelectorAll("button");

        buttons.forEach((button) => {
        button.addEventListener("click", () => {
            alert(button.id);
        });
        });

    Types of events:

    • click
      dblclick
      keydown
      keyup
